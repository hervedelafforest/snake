Application Web de Realité Augmentée

Accueil : Bienvenue sur notre application Web de réalité augmentée



La réalité augmentée est une technique qui superpose à la réalité une représentation numérique actualisée en temps réel.

Nous vous proposons de placer dans votre environnement n'importe quel modèle 3D par l'intermediaire de votre camera.

Vous trouverez dans le menu deroulant l'explication de comment fonctionne ce site ainsi que le catalogue ou vous retrouverez tout nos modeles 3D.
